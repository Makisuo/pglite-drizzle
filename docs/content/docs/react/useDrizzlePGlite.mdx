## Usage

Hook to get a drizzle db client instance.

```tsx
const db = useDrizzlePGlite()

await db.select().from(schema.users).where(eq(schema.users.id, 1))
```


### With Drizzle Options

```tsx
const db = useDrizzlePGlite({
	schema: schema,
})

await db.query.users.findMany({
	where: (table, {eq}) => eq(table.id, 1),
})
```
